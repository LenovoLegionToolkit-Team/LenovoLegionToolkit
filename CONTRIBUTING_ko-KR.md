## Lenovo Legion Toolkit 기여 가이드에 오신 것을 환영합니다!

### 이 기여 가이드의 다른 언어 버전:
* [English](CONTRIBUTING.md)
* [简体中文版开发者指南](CONTRIBUTING_zh-hans.md)
* [日本語版の貢献ガイド](CONTRIBUTING_ja-JP.md)

프로젝트에 기여할 시간을 내주셔서 감사합니다! LLT의 인기가 높아지면서, 기여가 원활하게 진행될 수 있도록 몇 가지 규칙을 안내드립니다.

<br/>

_이슈가 계속 늘어나고 있어 기준에 맞지 않는 이슈는 별도 경고 없이 삭제됩니다. 반복 위반자는 차단됩니다._

<br/>

**1. 이슈를 등록하기 전에 README를 먼저 확인해주세요**

[README](https://github.com/LenovoLegionToolkit-Team/LenovoLegionToolkit/blob/master/README.md)에는 자주 묻는 질문과 흔히 발생하는 문제에 대한 답변이 정기적으로 업데이트됩니다. 이슈를 등록하거나 디스커션을 시작하기 전에 먼저 내용을 확인해주세요.

**2. 이미 등록된 이슈를 확인해주세요**

이미 등록된 [이슈](https://github.com/LenovoLegionToolkit-Team/LenovoLegionToolkit/issues?q=is%3Aissue)와 [디스커션](https://github.com/LenovoLegionToolkit-Team/LenovoLegionToolkit/discussions?discussions_q=)을 먼저 확인해주세요. 중복되는 이슈나 디스커션은 등록하지 말아주세요. 이미 닫힌 이슈라도 댓글은 남길 수 있습니다. 알려진 문제는 **공식 디스코드 서버**에서도 논의할 수 있습니다.

**3. 영어를 사용해주세요**

모두가 대화를 따라가기 쉬워집니다.

**4. 프로젝트의 범위를 존중해주세요**

이 프로젝트는 모든 걸 다 해결해주는 만능 애플리케이션이 아닙니다. 프로젝트의 목표는 명확합니다: Legion 노트북을 위한 Vantage의 대체품을 제공하는 것입니다. 다른 종류나 모델의 기기에 대한 지원을 요청하지 말아주세요.

**5. 이슈를 등록하기 전에 문제를 검증해주세요**

정말로 LLT의 버그인지 먼저 확인해주세요 - 여기는 무료 시스템 문제 해결 포럼이 아닙니다. Windows를 수정해서 쓰고 있거나 Windows 자체가 이상하게 동작한다면, 그건 별개의 문제입니다.

**6. 문제를 최대한 자세히 설명해주세요**

좋은 설명이 핵심입니다. 이슈를 등록할 때 로그를 포함해 양식의 모든 항목을 채워주세요. 설명이 자세할수록 서로를 정확히 이해할 가능성이 높아집니다.

**7. 이슈와 디스커션에 좋은 제목을 붙여주세요**

그래야 목록을 훑어보거나 이슈와 디스커션을 따라가기가 훨씬 쉬워집니다. "LLT 사용 중 오류 발생"은 좋은 제목이 아닙니다.

**8. 주제에서 벗어나지 말아주세요**

불필요한 댓글이나 스팸을 남기지 말아주세요.

**9. 이슈 하나에 문제 하나**

하나의 이슈에서 여러 문제를 보고하거나 여러 기능을 요청하지 말아주세요. 문제, 주제, 아이디어마다 별도의 이슈나 디스커션을 만들어주세요. 그래야 추적이 쉬워집니다.

**10. 번역**

번역 기여는 [Crowdin](https://crowdin.com/project/lenovolegiontoolkit-unofficial)을 통해 이루어지므로, 기여하고 싶다면 해당 프로젝트의 접근 권한을 요청해주세요.

**11. 풀 리퀘스트**

풀 리퀘스트는 언제나 환영합니다. 아주 간단하고 이해하기 쉬운 PR이 아니라면, 먼저 이슈를 등록해 해결하려는 문제를 설명해주세요. 프로젝트의 방향과 맞지 않아 거절될 아이디어에 시간을 쏟는 건 의미가 없으니까요. 프로젝트의 코드 스타일과 아키텍처를 따라주세요. 복잡한 변경 사항은 먼저 **공식 디스코드 서버**에서 논의하는 걸 권장합니다.

<br/>

다시 한번, LLT를 더 좋은 프로젝트로 만드는 데 시간을 내주셔서 감사합니다!
